## 📝 測試環境配置

### 系統要求
- Node.js v14+
- npm v6+
- Apache 2.4+
- PHP 7.4+

### 安裝步驟
```bash
# 1. 安裝依賴
npm install

# 2. 啟動 XAMPP
# macOS: sudo /Applications/XAMPP/xamppfiles/bin/startlamp
# Linux: sudo /opt/lampp/lampp start
# Windows: 從 XAMPP 控制面板啟動

# 3. 運行測試
bash run-tests.sh          # macOS/Linux
bash run-tests.zsh         # zsh shell
run-tests.bat              # Windows
```
